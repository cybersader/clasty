# Welcome to Obsidian-in-Browser

This is a demo vault for testing the CRDT sync system.

## How It Works

1. **Browser Access**: You're viewing Obsidian through a VNC session in your browser
2. **CRDT Sync**: Changes sync automatically between all connected users
3. **No Conflicts**: Yjs CRDT handles simultaneous edits

## Try It Out

1. Open this vault in another browser tab
2. Edit this file in one tab
3. Watch the changes appear in the other tab
